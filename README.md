Run in development mode (Port 3000)

```
  npm run start
```

### Build for production

```
npm run build
```

### Run tests

- ### Linter Tests

To run tests, run the following command:

To check for html errors run:
```sh
  npx hint .
```
To check for css errors run:
```sh
  npx stylelint "**/*.{css,scss}"
```
To check for js errors run:
```sh
  npx eslint "**/*.{js,jsx}"
```
